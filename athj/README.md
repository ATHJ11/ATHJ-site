# ATHJ 公式サイト

ATHJ（Athletic Training Hub Japan）の公式1ページサイトのソースコードです。
シンプルで軽量な構成（HTML/CSS/Vanilla JS）で作成されており、ビルド不要で動作します。

## 📁 構成
- `index.html`: サイトのメインファイル（コンテンツ）
- `styles.css`: デザイン・レイアウト定義
- `script.js`: インタラクション（現在は最小限）

## 🚀 使い方（閲覧方法）
このフォルダにある `index.html` をダブルクリックしてブラウザで開くだけで閲覧可能です。ローカルサーバーは必須ではありません。

## ✏️ 編集箇所
以下の箇所は、実際の運用に合わせて変更してください。

### 1. 登録フォームのURL
`index.html` 内を検索し、以下の `href` をGoogleフォームのURLに書き換えてください。
```html
<a href="YOUR_GOOGLE_FORM_URL" target="_blank" ...>
```
```
※ファイル内に2箇所あります（ヘッダーとメインCTA）。

### 1-2. 登録者数APIの設定
トップページに登録者数を表示する場合、`script.js` の以下の変数を変更してください。
```javascript
const COUNT_API_URL = "https://script.google.com/macros/s/.../exec"; // あなたのGAS WebアプリURL
```
※JSON形式 `{ "count": 123 }` を返すGETリクエスト対応のAPIが必要です。未設定の場合は「登録者数：—人」と表示されます。


### 2. お問い合わせメールアドレス
`index.html` のフッター部分を修正してください。
```html
<a href="mailto:contact@athj.example">contact@athj.example</a>
```

### 3. メタデータ
`index.html` の `<head>` 内にある `title` や `meta description` を必要に応じて調整してください。

## 🌐 公開方法（GitHub Pagesの場合）
1. GitHubリポジトリを作成し、これらのファイルをアップロードします。
2. リポジトリの `Settings` > `Pages` を開きます。
3. `Branch` を `main` (または `master`) に設定し、フォルダを `/(root)` にして Save します。
4. 数分後、発行されたURLでサイトが公開されます。

## 🎨 デザインの調整
色は `styles.css` の冒頭にある変数（`:root`）を変更することで簡単に調整できます。
- `--color-primary`: メインカラー（現在はネイビー）
- `--color-accent`: アクセントカラー（現在はゴールド系）

ATHJとは

ATHJは、スポーツ現場で活動する専門職がつながり、実務的な課題解決と価値創造を行うためのネットワークです。
個々の現場で培われた知見を共有し、社会に還元していくためのハブとして機能します。

Mission / Vision / Values

Purpose

現場の知を、社会の価値へ。
スポーツの安全と専門性を、標準にする。

Mission

現場のAthletic Trainingの価値を言語化し、社会実装する。

Vision

すべてのスポーツ現場に、適切な安全とパフォーマンス支援が届く社会。

Values

  • Respect: 互いの専門性と背景を尊重する
  • Field-First: 机上の空論ではなく、現場の事実に即する
  • Collaboration: 既存の枠組みを超えて協働する

参加対象

スポーツ現場のAthletic Training業務に関心・従事する人

  • 実務者（Practitioner）： 現在、スポーツ現場でAthletic Training業務に従事している専門職（JSPO-AT/ATCに限定せず、PT・鍼灸師・柔整師等を含む）
  • 目指す人（Aspiring / Future Practitioner）： 将来スポーツ現場での実務に関わりたい人（学生・社会人を含む）

※目指す人は、相談・求人・報酬など一部領域で制限が入る場合がある（安心して情報共有できる環境を優先）、また給与サーベイ等は対象外となる場合があります。

重要な定義とスタンス

“AT”の定義について

ATHJが使用する「AT」という言葉は、特定の資格名ではなく「Athletic Training」という業務領域を指します。

活動のスタンス

本団体は、独自の資格付与や免許認定を目的とするものではありません。
既存の制度や職能団体を深く尊重し、それらがカバーしきれない実務領域を補完的に支える活動を行います。

ATHJが取り組むこと

  • 📊

    給与・働き方の実態を可視化（匿名サーベイ）

    実務者の協力で、リーグ/現場区分別の給与レンジや働き方を匿名で集計し、傾向として公開します。個人や組織が特定されない形でのみ扱います。

  • 🗣️

    AT業務の価値を“雇う側に伝わる言葉”にする（価値の言語化）

    現場で何が改善するのか（安全・欠場/再発・運営負担など）を、雇用側・社会に伝わる形に整理して発信します。

  • 📝

    仕事の型を共有する（テンプレ・手順の整備）

    職務内容の説明資料、緊急対応の流れ、連絡フロー、チェックリストなど「現場で使える型」を整えて共有します。

  • 🤝

    実務者同士のつながりを作る（相談・相互支援）

    同じ立場の実務者が相談・情報交換できる環境をつくります（運用方法は継続可能な形で検討）。目指す人は一部領域で制限が入る場合があります。

  • ⚖️

    役割と連携を整理する（多職種の境界を尊重）

    職種を超えて協働しやすいように、役割の線引きと連携の考え方を整理し、現場で使える形にします。

  • 🔭

    （将来）条件が見える求人・機会を増やす

    報酬レンジや役割が明確な機会が増えるよう、透明性の高い情報提供の仕組みを検討します。

  • サーベイ等は匿名で実施し、登録情報と回答を紐づけません（個人特定防止を徹底）。

ATHJが考えるAT（Athletic Training）

ATHJは資格団体ではありません。現場の安全を支える「役割と責任」を共有し、特定や対立を避けながら、データと言葉で現場の価値を社会へ届けます。

ATHJにおけるATの定義

ATHJにおけるAT（Athletic Training）は、資格名ではなく「役割と責任」で定義します。
予防・準備・救急対応・健康支援・多職種連携を通じて、スポーツ現場の安全と専門性を支える業務領域です。

ATHJがしないこと（境界線）

  • 免許・法令の範囲を超える医療行為は扱いません

    診断・処方・侵襲的処置など、各資格の範囲外の医療類似行為の推奨/指示はしません。

  • 特定（個人・選手・チーム・組織）につながる情報共有はしません

    相談は可能ですが「特定禁止」が前提です（伏せる、一般化、条件のぼかし）。

  • 資格の優劣を決めません／資格付与をしません

    JSPO-AT/ATC/各職種の序列化、格付け、認定ビジネスは行いません。

  • 特定の組織への肩入れ・介入をしません（共存の核）

    他団体の活動領域に踏み込み、対立を生む動きや政治的な介入は行いません。

  • 金銭・利害のトラブルを生む運用をしません

    交渉代行、斡旋、個別案件の仲裁ビジネス化、勧誘・営業目的の投稿は行いません。
